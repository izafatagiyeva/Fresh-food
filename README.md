# Fresh-Food
## About The Project
This is a modern, clean, and fully responsive website template built from scratch. The main focus of this project is to create a seamless user experience across various screen sizes (mobile, tablet, desktop) using the Bootstrap framework and custom CSS styling.
## Features
* **Fully Responsive:** Adapts beautifully to smartphones, tablets, and desktops using Bootstrap's mobile-first fluid grid system.
* **Modern UI Components:** Includes a responsive navigation bar, a hero carousel, interactive cards, and a functional footer.
* **Custom Styling:** Enhanced with custom CSS3 styles, transitions, and hover effects for a more engaging user experience.
* **Cross-Browser Compatible:** Tested and optimized to work smoothly across all modern web browsers (Chrome, Edge, Firefox, Safari).
## Built With
* **HTML5:** Semantic markup structure.
* **CSS3:** Custom styles, layout adjustments, and smooth animations.
* **Bootstrap 5:** Advanced grid layout, pre-built responsive components, and utility classes.
* **Google Fonts & FontAwesome:** Modern typography and crisp vector icons.
